# GitHub Trophies

Dynamic GitHub stats card for your README. Enter any username, pick a theme.

## Demo

![GitHub Stats](https://github-trophies-rho.vercel.app/api/stats?username=Rhizobium-gits&theme=v1)

## Usage

```markdown
![GitHub Stats](https://github-trophies-rho.vercel.app/api/stats?username=YOUR_USERNAME)
```

## Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `username` | (required) | GitHub username |
| `theme` | `v1` | `v1` `v2` `v3` `v4` `v5` `v6` `v7` `v8` |

## Deploy your own

1. Fork this repo
2. Deploy to [Vercel](https://vercel.com)
3. (Optional) Set `GITHUB_TOKEN` env var for higher API rate limits
